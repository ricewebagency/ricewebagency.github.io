<INSTRUCTIONS>
## Project Rules

- Always place JavaScript in separate `.js` files under `src/js/`.
- Use Tailwind CSS only.
- Do not write custom CSS unless absolutely necessary. If required, ask first. Place approved custom CSS in separate `.css` files under `src/css/`.
- Never run build commands. The user runs `npm run dev`, which handles builds automatically.
- Don't apply changes to the docs folder. The editable code exists in the `src` folder. The docs folder automatically syncs via the `npm run dev` command, which the user will handle. 

## Encoding Rules

- Always use UTF-8 encoding.
- Never replace or alter valid characters (e.g. →, €, —, ✓).
- Do not convert special characters into `?` or other placeholders.
- Preserve all original characters exactly as written.

</INSTRUCTIONS>
